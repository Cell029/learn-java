## 1.算术运算符的自增自减

```java
//自增自减,需要注意的是运算符在前的先进行自运算,然后再赋值
a++
a--
++a
--a
================
int a = 10;
int b = 0;
b = a++;//b = 10,a = 11
================
int a = 10;
int b = 0;
b = ++a;//b = 11,a = 11
```

>需要注意的是,在变量自增或者自减的时候,编译器会把这个值存入一个临时变量,查看下例,

```java
int a = 10;
a = a++;
sout(a);//此时输出的 a 为10,因为自增符在后,所以a是先赋值
```
