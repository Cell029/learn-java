# 一.方法的定义

**语法格式:**

```
[修饰符列表] 返回值类型 方法名(形式参数列表) {
	方法体;
}
```

**修饰符列表:**

```
public static
```

**返回值类型:**

>可以是Java中的任何一种数据类型,如果方法不返回任何数据时,需要把返回值类型写成void,其他的类型都需要添加return来返回相对应的数据

>需要注意的是,当return后面不加数据的时候,这个return就充当结束方法的作用

```
public static void ... {
	方法体;
}
public static int ... {
	return int;
}
public static String ... {
	return String;
}
public static boolean ... {
	return true/false;
}
```
