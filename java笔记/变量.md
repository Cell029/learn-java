
>在Java中,变量必须先定义后使用
# 一.基本类型变量

```
在定义变量的时候可以手动给他一个初始值,看似是初始值,实际上算是一种赋值
int a = 10;
如果不手动给他初始值的话,Java就会自动给他赋值,这也就是为什么基本数据类型中每个数据类型都有自己的默认值
int a;
```

>变量可以被赋值,也可以赋值给其他变量,其实都是被赋值...

```
int a = 10;
int b = a;//b = 10
```

>不过需要注意的是,当没有手动给某个变量赋初始值的话,是不允许把这个变量赋给其他变量的
