
>一个类只能有一个实例,并提供一个访问接口

```
1.构造方法私有化
2.对外提供一个公开的静态方法
```

# 一.饿汉模式

```Java
public class Singleton {
    //类加载时就创建实例，天然线程安全
    private static final Singleton instance = new Singleton();
    //构造器私有化，外部不能自主创建对象
    private Singleton() {
    
    }
    //提供全局访问点
    public static Singleton getInstance() {
        return instance;
    }
}
```

>因为创建的对象是静态变量,所以整个类只能初始化一次,也就是只能创建一次对象
