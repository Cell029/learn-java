# 4. 字符型常量和字符串常量的区别

| 对比项        | 字符型常量 `'a'`       | 字符串常量 `"a"`    |
| ---------- | ----------------- | -------------- |
| 数据类型       | `char`（基本类型，2 字节） | `String`（引用类型） |
| 引号使用       | 单引号 `'`           | 双引号 `"`        |
| 长度限制       | 只能是一个字符           | 可以是 0 个或多个字符   |
| 存储位置       | 栈 / 常量池           | 字符串常量池（方法区）    |
| 是否可参与运算    | 可以                | 不能（拼接除外）       |
| 是否可以用作数组索引 | 可以                | 不行             |
| 是否可以为 null | 不能（`char` 是基本类型）  | 可以是 null       |
