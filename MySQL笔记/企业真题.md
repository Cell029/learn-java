
# 1. 用一条 sql 语句查询出每门课都大于 80 分的学生

| name | course | score |
| ---- | ------ | ----- |
| 张三   | 数学     | 85    |
| 张三   | 英语     | 90    |
| 李四   | 数学     | 75    |
| 李四   | 英语     | 88    |
| 王五   | 数学     | 92    |
| 王五   | 英语     | 95    |

>根据学生姓名分组可以拿到一个学生的所有课程，然后找出最低课程的分数，判断是否大于 80 分

```sql
select name from t_student group by name having min(score) > 80;
```
